# Random

## Example BJS code:

`Libs.Random.sendMessage(["hello", "Hi", "Good day!", "Welcome!"]);`

## Get random integer value beetwen 0 and 10:

`Libs.Random.randomInt(0, 10);`

## Get random float value beetwen 200 and 400

`Libs.Random.randomFloat(200, 400);`
